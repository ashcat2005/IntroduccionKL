  3. Prender el equipo y arrancar desde la USB Siempre que un computador está prendiendo, durante los primeros segundos, aparecen mensajes propios de la máquina que indican cómo entrar al BIOS, es decir la tecla indicada. Puede ser un Esc, Tab, F8, F12... Por ejemplo:

    • MacBook: Dejar presionada la tecla alt/option
    • Dell: Presionar varias veces la tecla F12
    • HP: Presionar varias veces la tecla F8

  4. Instalar Ubuntu

    Instrucciones:

    http://www.ubuntu.com/download/desktop/install-ubuntu-desktop

    4.1 Seleccionar el Idioma

    4.2 Instalar Ubuntu

    4.3 Cumplir tres requisitos básicos:

     - Estar conectado a Internet. 
 - Si es un portátil, estar conectado a una toma de corriente.
 - Tener al menos 6.4 GB de espacio libre en Disco Duro. (Realmente recomiendo dejarla a Ubuntu más de 100GB)

    4.4 Sugiero Descargar las actualizaciones mientras se instala.

    4.5 Sugiero Instalar el software de terceros.

    4.6 Usted puede escoger si desea instalar Ubuntu junto a Microsoft...

    Depende de Usted. No tengo ninguna razón para usar Windows. Para usuarios expertos: Seleccionar "Más Opciones"

    4.7 Asigne el espacio que desea para Ubuntu. Acá hay que tener especial cuidad, ya que es irreversible. Si usted asigna muy poco espacio y necesita más eventualmente, debe realizar estos pasos nuevamente, realizar un BackUp y asegurarse de no perder información. Esto puede ser muy tedioso, teniendo en cuenta que cada persona personaliza el computador, así que se necesitaría instalar TODO nuevamente. De manera análoga, puede que no le guste Ubuntu (si es el caso me gustaría saber porqué alejandro.cardenasa@konradlorenz.edu.co) y haya asignado espacio que requiera para otro sistema operativo. La nueva asignación de espacio puede tardar varios minutos.

    4.8 Seleccionar la ubicación geográfica.

    4.9 Seleccionar la distribución del teclado. Acá es importante reconocer realmente qué teclado tiene, para que después no sea muy terrible buscar caracteres como ^,*,[,],ñ,¬,\, etc... La opción "Detectar teclado" es bastante divertida y sirve, en primera aproximación, para seleccionar adecuadamente el teclado correcto.

    4.10 Crear un nombre para el equipo. Recomiendo utilizar algo decente, ya que se usará todo el tiempo en las líneas de comando. No muy largo, no muy corto...

    4.11 La instalación puede tardar varios minutos. Mientras tanto puede leer la guia GitHub para Dummies.

    4.12 Una vez la instalación haya terminado correctamente, reinicie el computador. Dependiendo de la opción escogida en 4.6 el inicio del computador será: Entrar directamente a Ubuntu o desplegará una lista para escoger el sistema operativo a utilizar.

    4.13 Dependiendo la configuración del computador debe retirar la USB apagado o durante el reinicio. Para que no vuelva a iniciar desde la USB.

    Nota: Si por alguna razón, pueden haber muchas, la instalación no funciona. Sugiero dos cosas: a) Busque el error que ocurra en la web, me sorprendería que no hubiera información. b) Escríbame un correo con los detalles del suceso, agregando la descripción del computador, marca, sistema operativo ya instalado, tipo de instalación escogida, etc.

  5. Si está dentro del 20% de la población (la estadística es mia) que instala Ubuntu por primera vez y le funciona: Felicitaciones. No sé porqué pero casi siempre pasa algo, bien sea por la arquitectura del computador, la versión, un problema de compatibilidad, etc. Puede pasar que, si instaló Ubuntu quitándole espacio al otro sistema operativo ya existente, cuando intente iniciar otro sistema operativo tarde más de lo normal y deba hacer unas verificaciones de disco primero, es normal. En este caso, cada vez que prenda el computador debe escoger el sistema operativo a usar.

Consideraciones Generales:

Existe por lo menos otra manera de instalar un sistema operativo de manera virtual. Lo que se hace es utilizar un software que virtualiza arquitecturas para instalar un sistema operativo sobre otro, conocido como el anfitrión. En nuestro caso sirve en ambas vías; se puede virtualizar Ubuntu sobre Windows y se puede virtualizar Windows sobre Ubuntu. Aconsejo realizar este tipo de instalación para casos muy específicos y cuando el sistema operativo virtualizado se use realmente poco. Por ejemplo: Yo virtualizaba Windows desde Ubuntu para instalar juegos,utilizar SPSS (cuando no estaba para otras plataformas (ahora SPSS corre Python nativamente)) y los Software de JOVE y Celestron. Realmente no es aconsejable en general, ya que si no se tiene un súper computador ambos sistemas operativos funcionarán lentamente, consumiendo cada uno bastantes recursos de máquina. Hasta donde sé VirtualBox funciona bastante bien y es Software Libre. (https://www.virtualbox.org)
